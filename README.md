### Basic Commands

To run the tests

```
$ python manage.py test
```

To run the server

```
$ python manage.py migrate
$ python manage.py runserver
```

To compile i18n messages

```
$ python manage.py makemessages -l zh_Hans
$ python manage.py compilemessages -l zh_Hans
```
